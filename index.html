Web Safe Fonts

There are many different types of fonts that websites use, in most cases each website owner would have a set of brand style rules, this would apply to both a font and a selection of colour swatches. Because not all browsers are set up to use all fonts that are chosen for use on websites, there are "Web safe" fonts that are used. These allow for a seamless look amoungst any browser and are built into most of developers repitoirs.

  • Arial
  • Verdana
  • Tahoma
  • Trebuchet MS
  • Times New Roman
  • Georgia
  • Garamond
  • Courier New
